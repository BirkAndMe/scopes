Scopes JavaScript

The Scopes JavaScript will give a couple of tools, which makes it easier to transition between breakpoints in your JavaScript.



    
Enabling JS

    
Enabling the Scopes JavaScirpt HTML side is done by simply including the [PATH]/scopes/js/scopes.js on your page.

    
<script src="js/scopes.js"></script>

    
To enable the JavaScript on the scss side of things, you need to include the scope-js mixin.

But it's important to only include it once, and after you've initiated all your scopes.

In this example I've dedicated a scss file to this (javascript.scss).

    
scss/javascript.scss

    
Outputs:

    
css/javascript.css
