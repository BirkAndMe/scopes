The 4 scss files.

We use 4 scss files for this example, two of them are simple importer files, one is the settings and the last one is the actual style.

Read the inline comments in the files for a more detailed description.

_base.scss

This file imports all the files needed, so every style scss file should import this file.

scss/_base.scss

_scopes_modules.scss

This file imports all the scopes modules you want to use. These modules could just as easily be imported in the _base.scss file, and you can skip this extra file, it's a matter of taste.

scss/_scopes_modules.scss
