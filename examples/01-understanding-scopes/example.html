Understanding scopes

The concept of scopes is using scopes in css, or a simple type of scoping.
If you're new to the concept of a scope, there's a large wiki article about it, but for the purpose of Scopes.scss you can think of it as a breakpoint (which is 99% of the usecases in Scopes.scss).

The idea is that every scope (breakpoint) has a different set of variables, the same variables or the same variables with different values (confused yet?).
